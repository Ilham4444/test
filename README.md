#  Tomato Chat – Test Technique Frontend

Ce projet a été réalisé dans le cadre d’un **test technique pour un stage Frontend**.  
Il s’agit d’une application de chat inspirée d’un assistant IA, développée avec **React, TypeScript et Tailwind CSS**.


##  Objectif du test

- Créer une interface de chat 
- Gérer l’état des messages côté frontend
- Implémenter une sidebar interactive
- Utiliser TypeScript de manière correcte
- Appliquer Tailwind CSS pour le design

##  Technologies utilisées

-  React
-  TypeScript
-  Tailwind CSS
-  Lucide-react (icônes)
-  Vite

##  Structure du projet

```txt
src/
├── components/
```

##  Installation et exécution

Cloner le dépôt :

```bash
git clone https://github.com/Ilham4444/test.git
```
 Accéder au dossier du projet :
 
 ```bash
cd test 
```
Installer les dépendances:
 ```bash
npm install
```
Lancer le projet
```bash
npm run dev
```
L’application sera disponible sur :

[http://localhost:5173](http://localhost:5173)

##  Fonctionnalités principales

- Interface chatbot moderne (React + Tailwind CSS)
- Sidebar avec ouverture / fermeture animée
- Envoi et affichage des messages utilisateur
- Réponse automatique de l’IA (simulation front-end)
- Design responsive
